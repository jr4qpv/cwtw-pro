『CWTW-Pro』について
====================
CWTW-Pro はモールス符号の練習用のWindowsプログラムです。
Rev3.xでオープンソースソフトウェアとし全機能フリーで利用できるようになりました。

実行ファイル
------------
　`Readme.txt`　　　… 一般説明（本ファイル）
　`License.txt`　　 … ライセンスについての説明
　`CWTW-Pro.txt`    … プログラム説明ドキュメント
　`CWTW-Pro.exe`    … プログラム本体
　`<DATA>`          … 受信練習モード(ﾃｷｽﾄﾌｧｲﾙ)用のサンプルファイルフォルダ

プログラムの起動方法
--------------------
上記の実行ファイル一式を `C:\CWTW-Pro\` フォルダなどにコピーし、
その中の `CWTW-Pro.exe` ファイルを実行。
